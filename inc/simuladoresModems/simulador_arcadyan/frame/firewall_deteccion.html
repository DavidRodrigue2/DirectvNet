Firewall >Detección de Intrusión

Cuando el recurso SPI (Stateful Packet Inspection) firewall está Habilitado, todos los paquetes de datos se pueden bloquear. Stateful Packet Inspection (SPI) soporta distintos tipos de aplicativos que están usando números de puertas dinámicas.  Para los aplicativos marcados en la lista que se cita a continuación, el dispositivo soportará el pleno funcionamiento.

El firewall del dispositivo puede bloquear ataques de hackers comunes, incluyendo IP Spoofing, Le Attack, Ping of Death, IP with zero length, Smurf Attack, UDP port loopback, Snork Attack, TCP Nulo scan, y TCP SYN flooding.
