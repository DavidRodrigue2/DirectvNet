Wi-Fi  > Wi-Fi Protected (WPS)

Wi-Fi Protected (WPS) es el método estándar de la industria para simplificar la configuración de seguridad y gestión de redes Wi-Fi. Ahora usted puede configurar fácilmente y conectarse a una red WPA-habilitada 802,11 con dispositivos WPS-certificados que utilizan métodos o Personal Information Number (PIN) o Push Button Configuration (PBC) . Dispositivos legacy sin WPS se pueden agregar a la red usando el método de configuración manual tradicional.
Wi-Fi Protected (WPS) >
Estado WPS > Habilitado

Método Personal Information Number (PIN)

Digite el PIN del dispositivo cliente y haga un clic "Inscribir". Enseguida, inicie el WPS en el dispositivo cliente “dele” es utilitario inalámbrico o aplicación WPS en 2 minutos
Colocar PIN de los Dispositivos Clientes
 

Si un registrador externo está disponible, usted también puede digitar el PIN del Router 4G en el registrador externo. Para modificar el PIN del Router 4G, haga un clic "Generar Nuevo PIN" o haga un clic "Recuperar PIN Estándar" para redefinir el PIN al estándar de fábrica.
Router PIN 4G
Método Push Button Configuration (PBC)
Presione y sujete el botón PBC en su Router 4G durante 3 segundos o haga un clic "Inicie PBC". Enseguida, inicie PBC en el dispositivo que usted desea Conectar al Router 4G en 2 minutos.
