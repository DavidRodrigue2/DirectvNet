Help information
    When the DNS setting is selected, you can enter the specified domain name server IP, by which to provide domain name resolution service. When not selected, broadband service provider will automatically assign.
    If the domain name server IP is wrong, it will cause the webpage not able to be open, we suggest keeping it the default.
