Asistentes y resumen

Los asistentes para la [Configuración básica] y la [Configuración rápida] se encuentran a la izquierda.

A la derecha se muestra un resumen de la información del sistema de la Device.


Configuración básica


Ejecutar el Asistente para la conexión a Internet (Configuración sencilla)

Este asistente analiza la conexión a Internet y conecta el equipo a Internet. Al ejecutar el Asistente para la conexión a Internet, la mayoría de los módems de cable conectan el equipo automáticamente a Internet. Es posible que deba introducir información de PPPoE en los módems de DSL para que se conecten a Internet. Si su conexión necesita una contraseña u otra información de PPPoE, téngala a mano. Si no tiene esta información, puede obtenerla de su ISP.


Wireless SSID & Channel(11n 300Mbps Mode)

Este asistente le permite establecer un SSID (nombre de la red) y elegir un canal inalámbrico para la red.


Cifrado de la red inalámbrica (WEP/AES)

Este asistente le permite seleccionar el tipo de cifrado (WEP/AES) que usará la red inalámbrica.


Internet


Juegos por Internet (redireccionamiento de puertos)

Este asistente le permite establecer una NAT estática para jugar a través de Internet.


Skype

Este asistente le permite habilitar la función de pasarela de Internet de UPnP. Skype necesita que esto funcione correctamente.


Velocidad de multidifusión inalámbrica

Este asistente establece la velocidad de multidifusión inalámbrica y el paso IPv6.


Otros

Actualizar el firmware de la Device

Este asistente instala una actualización del firmware de la Device.


Inicializar Device

Este asistente restablece la configuración predeterminada de fábrica de la Device.


Información de INTERNET

Method of acquiring a Puerto INTERNET Dirección IP is displayed.


Estado de conexión

Si están seleccionados los servidores PPPoE o DHCP, el sistema muestra el estado de conexión del Puerto INTERNET.


Botón [Liberar]

El sistema muestra este botón si se usa el servidor DHCP. Si se pulsa, libera la dirección IP obtenida a partir del servidor DHCP del ISP. Normalmente, esta operación finalizará la conexión a Internet.


Botón [Renovar]

El sistema muestra este botón si está seleccionado el servidor DHCP. Renueva la dirección IP volviendo a obtener una dirección IP del servidor DHCP. En general, será la misma dirección IP que el servidor DHCP le proporcionó anteriormente.


Botón [Iniciar]

El sistema muestra este botón si está seleccionado el servidor PPPoE.
Este botón inicia una conexión continua y pasa a estado de espera una conexión a petición.


Botón [Conectar]

El sistema muestra este botón si está seleccionado el servidor PPPoE. Inicia una conexión a petición o manual. La conexión a petición se conecta automáticamente al iniciar la comunicación, suele ser necesario hacer clic en [Conectar].


Botón [Desconectar]

El sistema muestra este botón si está seleccionado el servidor PPPoE. Desconecta una conexión a petición o manual. Al desconectar una conexión a petición, se transforma en [Inactiva]. Al desconectar una conexión manual, se transforma en [Detenida].


Botón [Detener]

El sistema muestra este botón si está seleccionado el servidor PPPoE. Detiene conexiones continuas o a petición.


Botón [Comprobar conexión]

El sistema muestra la página que confirma la conexión a la INTERNET.


Botón [Actualizar]

Al hacer clic en este botón, el apartado Asistentes y resumen visualizado se renueva con la información actual.
